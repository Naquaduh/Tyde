AngularJS {ES6} // Bootstrap {Sass}

A complete, yet simple, starter for AngularJS and Bootstrap using webpack.

This workflow serves as a starting point for building Angular 1.x/Bootstrap 3.x applications using webpack. Should be noted that apart from the pre-installed angular package, this workflow is pretty much generic.

  • Heavily commented Webpack configuration with reasonable defaults.
  • AngularJS Application.
  • ES6 support with Babel.
  • Bootstrap 3 using Sass.
  • Lazy loading of Angular modules using ocLazyLoad.
  • Image optimization using webpack image-loader.
  • Source maps included in all builds. CSS Source maps are omitted from development build for performance concerns.
  • Development server with hot module reload.
  • Production builds with cache busting.
  • Testing environment using karma to run tests and jasmine as the framework. Not currently working...
  • No gulp or grunt, just npm scripts.
